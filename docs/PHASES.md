# 粒子弓箭世界 — 分阶段开发计划

## 核心理念

横屏 2D 世界（类泰拉瑞亚），所有物品由**最小粒子**构成。玩家以**剖面视角**观察世界——能看到物体内部的粒子排列与材质分层，而非简单轮廓勾勒。

| 材质 | 粒子颜色 | 物理特性 |
|------|----------|----------|
| 弓身（木质） | 棕色系 | 刚性约束，轻微弹性 |
| 弓弦（纤维） | 白色 | 柔性弹簧链，高张力 |
| 箭杆（木质） | 棕色系 | 刚性杆体 |
| 箭头（金属） | 灰色 | 高密度，穿透力强 |
| 箭羽（羽毛） | 浅色 | 低密度，空气阻力 |
| 靶体（草垛/木芯） | 黄绿/棕色 | 可穿透，粒子可脱落 |

---

## 阶段一：粒子引擎与像素渲染（当前 ✅）

**目标**：搭建最小可运行框架。

- [x] Canvas 像素风渲染器（整数缩放、无抗锯齿）
- [x] 粒子数据结构：`位置、速度、颜色、材质、固定标记`
- [x] 约束系统：刚性约束（弓身）+ 弹簧约束（弓弦）
- [x] Verlet 积分物理步进
- [x] 地面与基础横屏场景

**验收**：屏幕上可见棕色粒子组成的地块，白色粒子可独立运动。

---

## 阶段二：弓的粒子化建模

**目标**：用粒子精确表达弓身材质与弓弦材质。

- [x] 弓臂：双层棕色粒子弧形排列，模拟木材纤维方向
- [x] 握把：加粗粒子簇，固定于地面
- [x] 弓弦：白色粒子链，两端锚定在弓梢
- [x] 弓梢：粒子加粗，弦锚点
- [ ] 弓臂受力弯曲（拉弓时弓臂微弯）— 后续迭代

**验收**：固定在地上的弓，弓身棕色、弓弦白色，剖面清晰可见粒子排列。

---

## 阶段三：拉弓交互

**目标**：模拟真实拉弓操作。

- [x] 鼠标/触摸拖拽弓弦中部
- [x] 拉距限制与张力计算
- [x] 拉弓时弦粒子跟随，弹簧约束传递张力
- [x] 张力指示器 UI
- [ ] 弓臂随拉力弯曲变形 — 后续迭代

**验收**：玩家可拖拽弓弦，感受张力变化，松手后弦回弹。

---

## 阶段四：箭的粒子化与发射

**目标**：箭由粒子组成，被弓弦弹射出去。

- [x] 箭杆（棕色粒子链）+ 箭头（灰色粒子簇）+ 箭羽（白色粒子）
- [x] 搭箭：拉弓时箭附着在弦上
- [x] 发射：松手时根据拉距计算初速度
- [x] 飞行物理：重力、空气阻力
- [ ] 箭在飞行中旋转 — 后续迭代

**验收**：拉弓 → 松手 → 箭飞出，轨迹受重力影响。

---

## 阶段五：箭靶剖面与命中

**目标**：箭靶由粒子构成，支持剖面观察与穿透。

- [x] 靶体：外层草垛（黄绿粒子）+ 内层木芯（棕色粒子）
- [x] 靶环：同心圆粒子环（红/白交替）
- [x] 碰撞检测：箭粒子与靶粒子
- [x] 命中后箭嵌入靶体，靶粒子脱落
- [x] 得分系统

**验收**：箭射中靶体，可见剖面穿透效果，粒子散落。

## 技术架构

```
index.html
├── css/style.css          # 像素风 UI
└── js/
    ├── main.js            # 入口、游戏循环
    ├── renderer.js        # 像素渲染器
    ├── particle.js        # 粒子 + 约束
    ├── physics.js         # 物理世界
    ├── bow.js             # 弓粒子生成
    ├── arrow.js           # 箭粒子生成
    ├── target.js          # 靶粒子生成
    └── input.js           # 输入处理
```

# 粒子弓箭世界 — 分阶段开发计划

## 核心理念

横屏 2D 世界（类泰拉瑞亚），所有物品由**最小粒子**构成。玩家以**剖面视角**观察世界——能看到物体内部的粒子排列与材质分层，而非简单轮廓勾勒。

| 材质 | 粒子颜色 | 物理特性 |
|------|----------|----------|
| 弓身（木质） | 棕色系 | 刚性约束，轻微弹性 |
| 弓弦（纤维） | 白色 | 柔性弹簧链，高张力 |
| 箭杆（木质） | 棕色系 | 刚性杆体 |
| 箭头（金属） | 灰色 | 高密度，穿透力强 |
| 箭羽（羽毛） | 浅色 | 低密度，空气阻力 |
| 靶体（草垛/木芯） | 黄绿/棕色 | 可穿透，粒子可脱落 |

---

## 阶段一：粒子引擎与像素渲染（优化中 🔄）

**目标**：搭建最小可运行框架，确立横屏剖面视角表达规范。

- [x] Canvas 像素风渲染器（整数缩放、无抗锯齿）
- [x] 粒子数据结构：`位置、速度、颜色、材质、固定标记、所属物体`
- [x] 约束系统：刚性约束（弓身）+ 弹簧约束（弓弦）
- [x] Verlet 积分物理步进
- [x] 地面与基础横屏场景
- [x] 粒子按物体分层渲染（靶 → 弓 → 弦 → 箭）
- [x] 剖面标记与迎箭面方向指示

**剖面视角规范**：
- 弓：侧视可见木质年轮截面，弓弦在弓腹侧（背向靶标）
- 靶：靶面朝向弓箭（-X），玩家看到靶体纵向剖切的透镜形截面

**验收**：弓身呈弧形木质年轮层，箭靶为面向弓的剖面透镜形，非正面圆盘。

---

## 阶段二：弓的粒子化建模（重做 ✅）

**目标**：用粒子精确表达弓身材质与弓弦材质。

- [x] 弓臂：贝塞尔弧 + 法向年轮层（5 层棕色粒子/截面）
- [x] 握把：加粗矩形截面粒子簇，底部固定于地面
- [x] 弓弦：白色粒子链，位于弓腹侧（-X），锚定弓梢
- [x] 反曲弓形：弓梢向靶标方向（+X）偏转
- [ ] 弓臂受力弯曲（拉弓时弓臂微弯）— 后续迭代

**验收**：固定弓的剖面可见木质年轮层，弓弦白色且在弓腹侧。

---

## 阶段三：拉弓交互

**目标**：模拟真实拉弓操作。

- [x] 鼠标/触摸拖拽弓弦中部
- [x] 拉距限制与张力计算
- [x] 拉弓时弦粒子跟随，弹簧约束传递张力
- [x] 张力指示器 UI
- [ ] 弓臂随拉力弯曲变形 — 后续迭代

**验收**：玩家可拖拽弓弦，感受张力变化，松手后弦回弹。

---

## 阶段四：箭的粒子化与发射

**目标**：箭由粒子组成，被弓弦弹射出去。

- [x] 箭杆（棕色粒子链）+ 箭头（灰色粒子簇）+ 箭羽（白色粒子）
- [x] 搭箭：拉弓时箭附着在弦上
- [x] 发射：松手时根据拉距计算初速度
- [x] 飞行物理：重力、空气阻力
- [ ] 箭在飞行中旋转 — 后续迭代

**验收**：拉弓 → 松手 → 箭飞出，轨迹受重力影响。

---

## 阶段五：箭靶剖面与命中（重做 ✅）

**目标**：箭靶由粒子构成，剖面观察与穿透。

- [x] 靶体：透镜形剖面（中部厚、上下渐薄），材质按深度/高度分层
- [x] 靶面朝向弓箭（-X 迎箭面），非面向玩家
- [x] 靶环：剖面中呈水平条带（红心/草垛/木芯交替）
- [x] 碰撞检测：基于剖面体积（透镜形）
- [x] 命中后箭嵌入，靶粒子脱落
- [x] 得分系统（按剖面高度环带计分）

**验收**：箭靶呈侧视剖面透镜形，箭从左侧射入迎箭面。

## 技术架构

```
index.html
├── css/style.css          # 像素风 UI
└── js/
    ├── main.js            # 入口、游戏循环
    ├── renderer.js        # 像素渲染器
    ├── particle.js        # 粒子 + 约束
    ├── physics.js         # 物理世界
    ├── bow.js             # 弓粒子生成
    ├── arrow.js           # 箭粒子生成
    ├── target.js          # 靶粒子生成
    └── input.js           # 输入处理
```

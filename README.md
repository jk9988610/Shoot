# 粒子弓箭世界 (Particle Archery World)

横屏 2D 像素风游戏原型。世界中所有物品由**最小粒子**构成，以**剖面视角**观察粒子材质与内部结构。

## 快速开始

直接用浏览器打开 `index.html`，或使用本地服务器：

```bash
python3 -m http.server 8080
# 访问 http://localhost:8080
```

## 操作

- **拖拽弓弦**拉弓（白色粒子组成）
- **松开鼠标**发射箭矢
- 箭矢由棕色杆身 + 灰色箭头 + 白色箭羽粒子构成
- 射中粒子靶获得得分

## 材质粒子

| 颜色 | 材质 |
|------|------|
| 棕色 | 木质（弓身、箭杆） |
| 白色 | 纤维（弓弦）、羽毛（箭羽） |
| 灰色 | 金属（箭头） |
| 黄绿 | 草垛（靶体外层） |
| 红色 | 靶环 |

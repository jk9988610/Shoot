# 粒子弓箭世界 (Particle Archery World)

横屏 2D 像素风游戏原型。世界中所有物品由**最小粒子**构成，以**剖面视角**观察粒子材质与内部结构。

**https://jk9988610.github.io/Shoot/**

弓身绘制工具：**https://jk9988610.github.io/Shoot/bow-editor.html**

## 弓身绘制工具

打开 `bow-editor.html` 可自定义绘制弓身粒子剖面：

1. 用**画笔**绘制棕色木质粒子
2. 用**固定**标记握把区域（锚点处，游戏内固定于地面）
3. 设置**上梢**、**下梢**（弓弦挂载点）
4. 设置**弦位**（弓弦 X 坐标）
5. 点击**复制**或**下载**，将导出的 `custom-bow-data.js` 发给开发者集成

坐标以握把锚点 `(0,0)` 为原点，`dy` 向上为负，与游戏内弓身坐标一致。

## 快速开始

直接用浏览器打开 `index.html`，或使用本地服务器：

```bash
python3 -m http.server 8080
# 访问 http://localhost:8080
```

## 操作

- **拖拽弓弦**拉弓（白色粒子组成）
- **松开鼠标**发射箭矢
- 箭矢由棕色杆身 + 灰色箭头 + 白色箭羽粒子构成
- 射中粒子靶获得得分

## 材质粒子

| 颜色 | 材质 |
|------|------|
| 棕色 | 木质（弓身、箭杆） |
| 白色 | 纤维（弓弦）、羽毛（箭羽） |
| 灰色 | 金属（箭头） |
| 黄绿 | 草垛（靶体外层） |
| 红色 | 靶环 |
